## Tech Stack

- [TypeScript](https://www.typescriptlang.org/) - my main programming language.
- [React.js](https://reactjs.org) - UI creation.
- [Tailwind](https://tailwindcss.com/) and [PostCSS](https://postcss.org/) -
  styling.
- [Next.js](https://nextjs.org/) and [Remix](https://remix.run/) - frameworks.
- [EsLint](https://eslint.org/) and [Prettier](https://prettier.io/) - code
  linting & formatting.
- [ESBuild](https://esbuild.github.io/) - javascript bundler.
- [Prisma](https://www.prisma.io/) - ORM for Node.js.
- [Jest](https://jestjs.io/) - unit testing.
- [React Testing Library](https://testing-library.com/react) - React testing.
- [Cypress](https://www.cypress.io/) - integration and E2E testing.
- [MSW](https://mswjs.io/) - API mocking.

## Editor

[VSCode](https://code.visualstudio.com) it's my IDE/code editor.

## Services

- [Google Analytics](https://analytics.google.com) - tracking.
- [GitHub](https://github.com) this is where I host my code. I also run CI/CD
  pipelines with [GitHub Action](https://github.com/features/actions) .
- [Fly.io](http://fly.io/) and [Netlify](https://www.netlify.com/) are my
  deployment platforms.
- [Cloudinary](https://kcd.im/cloudinary) is my image hosting service.
- [Sentry](https://sentry.io) is my error reporting service.

## Tools

- [Chrome](https://www.google.com/chrome/) as my main browser.
- [Hyper](https://hyper.is/) as my terminal, outside VSCode.
- [GitHub](https://github.com) for my code repositories.
- [GitHub Mobile](https://github.com/mobile) to check GitHub on my phone.
